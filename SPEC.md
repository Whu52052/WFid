# 🕯️ 民国恐怖解谜游戏 - 《旧宅诡事》

## 1. Concept & Vision

**游戏定位：** 心理恐怖解谜游戏，背景设定在1920年代民国时期的废弃古宅。玩家扮演灵异调查记者，受邀前往调查一桩离奇的失踪案。古宅中弥漫着诡异的氛围，碎片化的日记、模糊的照片、残破的家具诉说着过去的故事。游戏强调心理压迫感而非Jump Scare，通过动态音效、画面抖动和开放式叙事营造持续的恐惧体验。

**核心体验：** 探索 → 调查 → 解谜 → 拼凑真相 → 多结局

**游戏特色：**
- 民国复古美学：泛黄纸张、老式家具、中式图案
- 心理恐怖：环境暗示、碎片叙事、开放式结局
- 多人合作：玩家可组队探索，各自调查不同线索后共享发现
- 逻辑推理谜题：从物品描述、笔记片段中推理密码/逻辑
- 深层隐喻：故事背后有深层含义和隐喻
- 持续更新：游戏内容将持续更新完善

## 2. Design Language

### 2.1 Aesthetic Direction
**风格参考：** 1920年代中国古宅 + 暗黑哥特 + 民国老照片质感

### 2.2 Color Palette
| 用途 | 颜色 | Hex |
|------|------|-----|
| 主背景 | 深红褐色 | `#1A0A0A` |
| 次背景 | 暗红 | `#2D1A1A` |
| 强调色 | 血红 | `#8B0000` |
| 文字主色 | 米黄 | `#D4C4A8` |
| 文字次色 | 暗金 | `#B8956C` |
| 边框/分割 | 深棕 | `#3D2B2B` |
| 危险提示 | 暗红 | `#660000` |

### 2.3 Typography
- **标题/场景名：** "ZCOOL KuaiLe" 或 "Ma Shan Zheng"（中文书法风）
- **正文/日记：** "LXGW WenKai" 或系统宋体
- **UI/按钮：** "Noto Sans SC"
- **诡异文字效果：** 使用不规则字间距模拟手写

### 2.4 Spatial System
- 场景采用左右分栏：左侧大地图/场景图，右侧文字描述和交互
- 移动端优先：单栏垂直布局，场景图在上，文字在下
- 间距使用 8px 基准网格

### 2.5 Motion Philosophy
- **画面抖动：** 场景转换、恐怖事件触发时轻微抖动（2-4px）
- **打字机效果：** 文字逐字显示，模拟手写/老旧打字机
- **渐入渐隐：** 场景切换使用 400ms 淡入淡出
- **异常闪烁：** 特定恐怖场景下画面短暂变暗/变红

### 2.6 Visual Assets
- **图标：** 简约线性图标，使用暗金色
- **场景图：** 像素风或手绘风格，配合CSS滤镜（模糊、噪点）增加年代感
- **emoji辅助：** 🕯️ 📜 🗝️ 🔍 👤 🚪 等用于选项和提示

## 4. Features & Interactions

### 4.1 核心功能

#### 4.1.1 场景探索
- 点击场景区域查看详情
- 选择"调查"选项探索当前场景
- 调查可能发现：物品、线索、笔记、隐藏区域
- 已调查区域标记为"已探索"（不同色调）

#### 4.1.2 物品系统
- 收集场景中的物品（钥匙、笔记、照片等）
- 物品可组合使用（例：钥匙+锁）
- 物品描述可能包含隐藏线索

#### 4.1.3 笔记系统
- 自动记录重要发现
- 笔记分类：📜 日记片段 | 🔍 调查线索 | 💬 对话记录
- 笔记可帮助解谜

#### 4.1.4 谜题类型
- **密码破译：** 从笔记/物品中找规律，推断密码
- **逻辑推理：** 根据日记时间线推断事件顺序
- **物品组合：** 组合多个物品触发事件
- **场景互动：** 在正确场景使用正确物品

#### 4.1.5 多人合作
- 创建房间获得房间码
- 邀请码分享给队友
- 各自探索不同区域（避免重复劳动）
- 共享发现：一人找到线索后其他人也可看到
- 实时队友位置显示

### 4.2 交互细节

| 操作 | 反馈 |
|------|------|
| 点击选项 | 按钮下沉效果 + 轻微震动 |
| 发现物品 | 物品图标弹入 + 收集音效 |
| 恐怖事件 | 画面抖动 + 背景变红闪烁 |
| 解谜成功 | 光芒效果 + 解锁音效 |
| 队友发现 | 通知气泡弹出 |

### 4.3 边界情况
- **网络断开：** 显示重连按钮，本地缓存进度
- **队友离线：** 标记为离线，但保留其发现
- **输入无效：** 提示"这里似乎没什么特别的"
- **卡关：** 提供"求助"按钮，消耗提示道具

## 5. Component Inventory

### 5.1 按钮组件
- **主按钮：** 深红背景 + 暗金边框 + 米黄文字
- **次按钮：** 透明背景 + 暗金边框
- **禁用态：** 灰暗色调 + 50%透明度
- **悬停态：** 边框发光效果
- **点击态：** 内凹阴影 + 轻微缩放

### 5.2 卡片组件（物品/笔记）
- **默认：** 暗红边框 + 半透明深红背景
- **悬停：** 边框变亮 + 轻微上浮
- **选中：** 强调色边框 + 背景加深
- **锁定：** 模糊遮罩 + 🔒 图标

### 5.3 场景组件
- **未探索：** 正常显示
- **已探索：** 灰暗滤镜 + ✓ 标记
- **当前：** 微光呼吸效果

### 5.4 对话框组件
- **系统提示：** 顶部滑入，3秒后自动消失
- **队友消息：** 底部气泡，支持emoji
- **解谜提示：** 居中模态框 + 暗色遮罩

### 5.5 输入组件
- **文本输入：** 泛黄纸张纹理背景 + 打字机字体
- **密码输入：** 星号遮罩 + 可显示切换
- **验证码：** 大写字母 + 扭曲效果防机器

## 6. Technical Approach

### 6.1 前端技术栈
- **框架：** React 18 + TypeScript
- **构建：** Vite
- **样式：** Tailwind CSS
- **动画：** Framer Motion
- **状态：** Zustand（本地） + React Query（服务器）
- **音效：** Howler.js
- **多端适配：** 响应式布局 + 移动端优先

### 6.2 后端技术栈
- **框架：** Express.js（Node.js）
- **数据库：** SQLite（Prisma ORM）
- **实时：** Socket.IO（多人合作）
- **认证：** JWT Token
- **部署：** PM2 + Nginx

### 6.3 数据模型

```
User

├── nickname: string

├── id: string (cuid)

├── id: string

├── currentSceneId: string
├── inventory: JSON (物品列表)
├── notes: JSON (已解锁笔记)
├── exploredScenes: JSON (已探索场景)

└── joinedAt: datetime

Scene
├── id: string
├── chapterId: number
├── name: string
├── description: string
├── imageAsset: string
├── connections: JSON (可前往的场景)
├── items: JSON (场景内物品)

├── id: string
├── sceneId: string (FK Scene)
```

## 7. Game Content Structure

### 7.1 章节概览（超长篇 - 15小时以上）

**序章：踏入古宅**
- 场景：古宅大门 → 前厅 → 客厅 → 餐厅
- 目标：进入古宅，了解背景
- 谜题：大门密码锁（从邀请函推理）
- 时长：约2-3小时

**第一章：过去的回响**
- 场景：书房 → 卧室 → 阁楼 → 主卧室
- 目标：找到主人的日记
- 谜题：日记密码（时间线推理）
- 时长：约3-4小时

**第二章：隐藏的真相**
- 场景：地下室 → 密室 → 后院 → 祠堂
- 目标：揭开古宅的秘密
- 谜题：组合谜题、最终密码
- 时长：约3-4小时

**第三章：轮回**
- 场景：古宅深处 → 未知空间 → 抉择之地
- 目标：面对真相，做出选择
- 谜题：最终抉择
- 时长：约2-3小时

**尾声：各结局**
- 根据玩家选择触发不同结局
- 时长：约1-2小时

### 7.2 结局列表（开放式结局 + 多结局）
1. **真相大白** - 揭开所有秘密，成功逃离
2. **迷失** - 被困在古宅中，永远无法离开
3. **牺牲** - 为了拯救他人牺牲自己
4. **共生** - 与古宅的存在达成某种协议
5. **轮回** - 成为古宅的下一个传说
6. **觉醒** - 打破轮回，释放所有被困灵魂
7. **继承** - 继承古宅的一切，成为新的守护者

## 8. Audio Design

### 8.1 背景音乐
- **正常探索：** 低频嗡鸣 + 偶尔木地板吱呀声
- **紧张时：** 节奏加快 + 低频脉冲
- **发现重要线索：** 短暂的静默后出现旋律片段
- **恐怖事件：** 不协和音 + 尖锐音效

### 8.2 音效
- 📜 翻页声
- 🗝️ 钥匙插入/转动
- 🚪 开门/关门
- 👣 脚步声（在不同地面材质）
- 💨 风/呼吸声
